# Genetic Algorithm for Tetris and Petris
![72 tetris](https://github.com/jdziecielski/genetic-algorithm-for-tetris/assets/106883711/b7d28fa3-f62f-40b6-b4d7-6d9fa893fc1c)

### The project consists of both playable and genetic version of two games - Tetris and Petris. The genetic version of the game serves a purpose of showing that a genetic algorithm can be used to solve Tetris or Petris. Petris is a version of Tetris game, in which all of the blocks are permutations of 5 connected blocks, where as the regular Tetris is using shapes consisting of 4 connected blocks. The project is implemented in python, using libraries: pygame and numpy.
